## Employee Contact Form Backend

    This is a api to<br>
    <ul>
        <li>Create Employee with multiple contact details (Relationship mapping)</li>
        <li>List Employee (with pagination)</li>
        <li>Update Employee</li>
        <li>Delete Employee</li>
        <li>Get Employee</li>
    </ul>



## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
